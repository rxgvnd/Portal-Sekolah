Login U-School:
Jika login sebagai admin dimana username '1000' dan password 'admin' maka page akan redirect ke halaman admin. Dimana halaman admin, bisa add user, view dan edit user, delete user, dan bisa ke coder's profile.

Jika login sebagai guru maka page akan redirect ke halaman guru. Dimana halaman guru bisa edit nilai mahasiswa.

Jika login sebagai murid maka page akan redirect ke halaman murid. Dimana halaman murid hanya bisa melihat ringkasan nilai serta grade dan dinyatakan lulus atau tidak.
